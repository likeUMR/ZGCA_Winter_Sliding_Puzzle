# 华容道小游戏调试指南

为了保持页面整洁，正式版移除了页面上的调试按钮。您可以通过浏览器控制台（F12）执行以下命令：

## 1. 立即通关
```javascript
win()
```
**功能**：将当前棋盘的所有棋子瞬间复原到正确位置，并触发通关结算画面。用于快速测试成就解锁和结算效果。

## 2. 清除所有进度
```javascript
clear()
```
**功能**：删除浏览器本地存储的所有通关记录（LocalStorage），并自动刷新页面。用于重新开始测试完整的游戏流程。

---

## 版本分支说明
- **release-v1** (当前): 正式版，无调试按钮，仅支持控制台命令。
- **8to4-rounds**: 调试版，页面底部带有显眼的“复原”和“清除进度”文字链接。
